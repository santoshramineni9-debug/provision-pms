🔀 Processing vs Adjudication

⚙️
Claim Processing
The ENTIRE Journey
  • Patient visit → Code assignment
  • Charge entry → Claim creation
  • Scrubbing → Submission
  • Tracking → Follow-up
  • Payment posting → Close

Who: Provider + Payer + AR Team

🔍
Adjudication
Insurance Decision ONLY
  • Check eligibility
  • Check coverage
  • Verify auth & network
  • Calculate payment
  • Decide: Pay / Deny / Partial

Who: Insurance/Payer ONLY

💡 Key Difference

Processing = Everything that happens  |  Adjudication = One step inside processing (the insurance review)

❌ Denial Type: Non-Covered Services (Patient)

Status Code: NCS - Non-Covered Services

Insurance denied the claim because the service is not covered under patient's plan. This could be a non-covered diagnosis or procedure.

💬
Questions to Ask Insurance
5 Key Questions
  • 1. May I have the denial date and claim number?
  • 2. Which services are not covered under this plan?
  • 3. Can we appeal with medical records? Get fax# or mailing address
  • 4. Does this patient have any other insurance?
  • 5. Could you fax me an EOB?
⚙️
Action to Take
Next Steps
  • Bill Secondary Payer if other insurance exists
  • Bill Patient if no other insurance available
Status Code: NCS
Non-Covered Services

💡 Easy Way to Remember

NCS = Patient's plan doesn't cover it
→ Check for other insurance → If none → Bill the patient

❌ Denial Type: Non-Covered Services (Provider)

Status Code: NCS - Non-Covered Services

Insurance denied because the service is not covered for this specific provider. Provider may be out-of-network or not credentialed for this service.

💬
Questions to Ask Insurance
4 Key Questions
  • 1. What is the denial date and claim number?
  • 2. Which services are not covered for this provider?
  • 3. Can we appeal with medical records? If yes, request insurance to reprocess
  • 4. Could you fax me an EOB?
⚙️
Action to Take
Next Steps
  • Escalate to Coding Team for review
  • Check if provider needs credentialing
  • May need corrected claim with different provider info
Status Code: NCS
Non-Covered Services (Provider)

💡 Easy Way to Remember

NCS (Provider) = Provider issue, not patient
→ Escalate to Coding → Check provider network status → Fix provider info → Resubmit
